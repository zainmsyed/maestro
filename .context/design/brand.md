# Brand

<!-- source: intake -->
<!-- Tone of voice, naming conventions, logo/asset notes, brand constraints. -->

## Tone
<!-- source: story-005 -->
- Calm, precise, operational, and understated.
- Copy should feel like a planning instrument: concise labels, clear state, no playful embellishment.

## Naming
<!-- source: story-005 -->
- Product name: Maestro.
- Navigation names should be short nouns: Roadmap, List, Health, Settings.
- Use “roadmap” for the imported planning corpus and “sync” for backend freshness/status language.

## Constraints
<!-- source: story-005 -->
- Dark mode first / dark mode only for current shell.
- External Google Fonts are allowed for DM Mono and Fraunces.
- Visual direction follows the intake HTML mockup: clean minimal dark UI, muted gold accent, mono data labels, Fraunces display headings.
- Spacing should be roomy rather than dense, except for persistent chrome.
